Алгоритм защиты:
1. Выбрать видео файл
2. Для выбранного видео файла составляется копирайт
3. Копирайт шифруется симметричным алгоритмом шифрования AES
4. В видео встраивается зашифрованный копирайт
5. Получается хэш от нового видео файла
6. Подписывается хэш - ЭЦП
В файл лицензии сохраняется:
- название нового видео файла
- дата создания
- дата модификации
- создатель
- хэш
- ЭЦП
- копирайт


Алгоритм проверки:
1. Выбирается видео файл
2.1. Получается хэш от файла
2.2. Подписывается шэш - новая ЭЦП
2.3. Полученная новая ЭЦП сверяется с имеющейся
3.1. Через процесс стеганографии получается копирайт из каждого кадра (по заложенному алгоритму)
3.2. Расшифровывается копирайт
3.3. Сверяется копирайт
4. Если все проверки пройдены успешно - лицензия файла сохранена
